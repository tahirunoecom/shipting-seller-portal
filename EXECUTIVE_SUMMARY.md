# Shipting Seller Portal - Executive Summary

## What Is This?

**Shipting is an all-in-one e-commerce platform that lets store owners sell products through WhatsApp, manage orders visually, and deliver using either their own team or a network of local delivery partners - all while administrators oversee quality and payments.**

---

## The Problem We Solve

1. **Customers prefer WhatsApp** - 2+ billion users check it daily, but most stores don't sell there
2. **Order management is messy** - Sellers juggle spreadsheets, notes, and messages
3. **Finding delivery help is hard** - Small stores can't afford full-time drivers
4. **Getting paid is complicated** - Manual payment tracking and bank transfers
5. **No oversight** - Platform owners need to ensure quality and prevent fraud

## How It Works (End-to-End)

### **Customer Places Order:**
1. Customer scans QR code or clicks link
2. Opens WhatsApp with seller's store
3. Browses product catalog in WhatsApp
4. Chats with bot, asks questions
5. Places order through chat

### **Seller Processes Order:**
1. Order appears in seller dashboard (with sound/notification)
2. Seller reviews items, customer address
3. Accepts order ‚Üí chooses delivery method
4. If self: Seller packs ‚Üí ships ‚Üí delivers ‚Üí marks complete
5. If driver: Nearby drivers notified ‚Üí one accepts ‚Üí picks up ‚Üí delivers

### **Driver Delivers Order:**
1. Sees available orders nearby
2. Accepts interesting order
3. Heads to store (app guides with steps)
4. Picks up items
5. Delivers to customer
6. Takes photo proof, gets signature
7. Completes delivery ‚Üí earns money

### **Seller Gets Paid:**
1. Views available earnings in billing page
2. Requests payout (enters amount or full balance)
3. Admin reviews request
4. Admin approves (full or partial)
5. Stripe automatically transfers to seller's bank
6. Money arrives in 2-7 days

### **Admin Oversees Everything:**
1. Reviews new seller applications ‚Üí approves/rejects
2. Views all sellers' stats, products, orders
3. Reviews payout requests ‚Üí approves/rejects
4. Ensures platform quality and safety
5. Can create manual payouts for special cases
